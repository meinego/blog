## 문제
0 < A, B < 10 자연수가 주어질 때, A+B를 구하는 문제

## 풀이
stdout으로 A+B를 출력하면 된다.

## 코드
```c++
#include <bits/stdc++.h>
using namespace std;
int a,b;

int main() {
    cin>>a>>b;
    cout << a+b;
    return 0;
}
```
